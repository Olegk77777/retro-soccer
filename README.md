# Футбол 98 (рабочее название)

Браузерный ретро-футбол 11×11 в эстетике конца 90-х: low-poly в духе PS1, CRT-фильтр с ТВ-пресетами, русский комментатор, сборные эпохи ЧМ-98/Евро-96. Реалистичный геймплей с сильной обороной — счета 1:0, а не 7:3.

*Retro football browser game in late-90s aesthetics: PS1-style low-poly, CRT filter, realistic low-scoring gameplay. Work in progress.*

**Статус:** в разработке, играть пока нечего — игра появится здесь по мере готовности (GitHub Pages).
