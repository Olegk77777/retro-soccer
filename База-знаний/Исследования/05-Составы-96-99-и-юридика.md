# Отчёт: данные составов 1996–99 и юридика реальных имён для ретро-футбола

## (а) Где брать составы 1996–1999

### Лучший первичный источник — английская Википедия
Проверено на живых страницах (июль 2026):

- **[1998 FIFA World Cup squads](https://en.wikipedia.org/wiki/1998_FIFA_World_Cup_squads)** — все 32 сборные по 22 игрока. Поля в таблицах: **номер (1–22), позиция (GK/DF/MF/FW), имя, дата рождения (возраст), матчи за сборную (caps), клуб**. Пример: Ronaldo — №9, FW, 22.09.1976 (21 год), 37 caps, Inter Milan. **Роста и голов в таблицах нет.**
- **[UEFA Euro 1996 squads](https://en.wikipedia.org/wiki/UEFA_Euro_1996_squads)** — те же 6 полей, заполнены по всем игрокам без пропусков (пример: Seaman — №1, GK, 24 caps, Arsenal).
- **Страницы клубных сезонов** вида [1998–99 Manchester United F.C. season](https://en.wikipedia.org/wiki/1998%E2%80%9399_Manchester_United_F.C._season) — есть раздел Squad statistics (матчи/голы игроков за сезон); полнота номеров варьируется от клуба к клубу, топ-клубы АПЛ/Серии А покрыты хорошо.

Практично: тянуть через Wikipedia API и парсить wikitable — это самый структурированный и полный источник; сами факты составов авторским правом не охраняются (в США — Feist), текст статей под CC BY-SA.

### Дополнительные источники (перекрёстная проверка)
- **RSSSF**: хаб [World Cup 1998 finals](https://www.rsssf.org/tables/98full.html) — ссылки на squads, бомбардиров, тренеров, резервистов; [Euro 1996](https://www.rsssf.org/tables/96e.html). Формат — плоский текст, парсить неудобно, но данные авторитетные. Хорош для сверки спорных случаев.
- **[Planet World Cup — 1998 squads](https://www.planetworldcup.com/CUPS/1998/wc98squads.html)** — постранично по сборным, удобная альтернатива.
- **[National-Football-Teams.com](https://www.national-football-teams.com/country/59/1996/England.html)** — есть страницы «страна+год» (England 1996 и т.п.) и карточки игроков (там обычно рост и позиция), но структура под ручной сбор, поля на страницах год-страна я автоматически подтвердить не смог — проверять вручную.
- **Transfermarkt** — исторические составы есть по URL-паттерну `.../kader/verein/<id>/saison_id/1997`, но сайт **блокирует автоматические запросы** (WebFetch отбился, проверено) — только ручной просмотр. По опыту сообщества номера за 90-е там часто неполные; рост/дата рождения — в карточках игроков.

**Вывод по данным:** номера, позиции, возраст, клуб — берём из Википедии; рост — либо вручную из Transfermarkt/NFT, либо не использовать. Рейтингов нигде нет — генерировать эвристикой (caps + уровень клуба + позиция).

### Предлагаемый JSON-формат команды

```json
{
  "id": "bra-1998",
  "displayName": "Бразилия",
  "shortName": "BRA",
  "year": 1998,
  "colors": { "home": "#FFDC02", "away": "#0033A0" },
  "formation": "4-4-2",
  "players": [
    {
      "name": "Роландо",
      "number": 9,
      "pos": "FW",
      "age": 21,
      "foot": "R",
      "skill": { "speed": 95, "shoot": 93, "pass": 84, "def": 35, "stamina": 88, "gk": 5 }
    }
  ]
}
```

Шкала skill 1–99 (привычно по PES/FIFA); `gk` держать у всех — упрощает логику замены вратаря. Позиции только 4 базовые (GK/DF/MF/FW) — как в источниках, без детализации, которой в данных 90-х всё равно нет.

## (б) Юридика

### Почему PES переименовывал
Права лежат в трёх слоях, и на каждый нужна отдельная лицензия:
1. **Имена/образы игроков** — личные права; в индустрии лицензируются коллективно через **FIFPro** (мандат на имя и образ игроков стран-членов профсоюза) либо напрямую через лиги/клубы; АПЛ продаёт видеоигровые права коллективно и эксклюзивно ([Push Square, 2020](https://www.pushsquare.com/news/2020/11/fifa_21_rights_row_rumbles_on_as_fifpro_agents_weigh_in), [EA Sports Law](https://www.easportslaw.com/news/easports-image-rights-fifa)).
2. **Названия/эмблемы клубов** — товарные знаки клубов/лиг.
3. **Формы** — дизайн/trade dress.

Где лицензий не было, Konami ставила «Man Red», «London FC» и фальш-имена игроков ([Goal.com](https://www.goal.com/en/lists/the-best-worst-fake-team-names-on-pes-pro-evo/anccdjz4ch841hslr7suaqk9j), [полный список PES 2020](https://www.gamesatlas.com/pes-2020/features/pes-2020-all-the-real-names-of-the-unlicensed-teams-and-leagues-of-efootball-pes-2020), [классические фальш-имена](https://playerscouture.com/blogs/news/classic-fake-name-football-players-rom-the-iss-pro-evolution-amp-pro-evolution-soccer-pes-games)).

### Ключевые прецеденты
- **Keller v. Electronic Arts (NCAA)** — EA использовала атрибуты реальных игроков (номер, рост, позиция) **даже без имён**; суд 9-го округа отклонил защиту по Первой поправке (2013), итог — мировое на **$60 млн** (2015) ([Hagens Berman](https://www.hbsslaw.com/cases/ncaa-video-games), [решение суда](https://rightofpublicity.com/pdf/cases/KellerVsEAruling7-31-13.PDF)). Вывод: «убрать имя, оставить всё остальное» — не панацея.
- **Manchester United v. Sega/Sports Interactive** (2018–2021) — клуб судился за само использование имени «Manchester United» в Football Manager; закончилось мировым без признания вины, SEGA заявила, что лицензия на имя не нужна, но по сообщениям имя в игре изменили жестом доброй воли ([LawInSport](https://www.lawinsport.com/topics/item/sega-s-battle-against-man-utd-in-football-manager-trade-mark-case-ends-in-settlement), [William Fry](https://www.williamfry.com/knowledge/manchester-united-settle-trade-mark-dispute-with-sega/)). Даже гиганту 3 года тяжбы.
- **Златан против EA/FIFPro (ноябрь 2020)** — игроки активно следят за использованием своих имён; права не истекают с завершением карьеры ([The Week](https://theweek.com/gaming/108772/zlatan-vs-ea-sports-image-rights-fifa-video-game)).
- **Фан-сцена PES-патчей** (SmokePatch, PESNewupdate, ultigamerz — живы и в 2025) существует 20+ лет; задокументированных массовых C&D от Konami против option-файлов я не нашёл. Секрет живучести: игра продаётся «чистой», а реальные имена накатывает **сам пользователь** отдельным файлом. Для сравнения — фан-игры по Nintendo регулярно убивают C&D-письмами ([Gameranx](https://gameranx.com/features/id/159002/article/10-incredible-fan-games-that-were-legally-taken-down/)).

### Риски бесплатной фан-игры и безопасная схема
«Бесплатно» ≠ «безопасно»: права на имя/образ и товарные знаки нарушаются независимо от заработка; реальный сценарий — не суд, а C&D-письмо с требованием убрать игру. Для хобби-проекта «в стол» риск мизерный, для публикации на CrazyGames/itch.io — заметный.

Безопасная схема (проверена индустрией, путь PES):
1. **Имена — изменённые, но узнаваемые**: Роналдо→Роландо, Зидан→Зидано (не использовать реальные фамилии целиком).
2. **Вымышленные названия и эмблемы** команд, свои цвета «по мотивам» без копирования дизайна формы и гербов.
3. **Никаких реальных лиц/фото** — только low-poly абстракция (PS1-стиль тут сам по себе защита).
4. **Реальные номера/позиции/возраст** оставить можно — это факты, но помнить урок Keller: чем больше совпадений с реальным человеком, тем тоньше лёд.
5. На будущее — **паттерн option-файла**: игра поставляется с вымышленными данными + встроенный редактор/импорт JSON, «реальный» датасет живёт отдельно как пользовательский контент.
