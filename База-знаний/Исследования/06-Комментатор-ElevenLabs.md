# Комментатор из пре-генерированных клипов ElevenLabs — отчёт

## а) Как это делали игры 90-х

**FIFA 96–98 (EA Sports).** FIFA 96 первой в серии получила полноценный play-by-play комментарий Джона Мотсона (в версиях на движке Virtual Stadium), в FIFA 97–98 добавили аналитика Энди Грея и ведущего Дэса Лайнема ([Wikipedia: FIFA 97](https://en.wikipedia.org/wiki/FIFA_97), [IMDb](https://www.imdb.com/title/tt1779400/characters/nm0609441)). Точное число фраз EA не публиковала, но известны две ключевые вещи ([VGFacts](https://www.vgfacts.com/game/fifasoccer971/), [Wikipedia](https://en.wikipedia.org/wiki/FIFA_97)):
1. **Каждую фразу Мотсон записывал 4 раза с разной интонацией** — чтобы не звучало роботом.
2. **Фамилии большинства игроков — отдельные клипы**, которые движок вклеивал в «болванки» фраз. Именно отсюда фирменные баги эпохи: комментатор называл не того игрока и путал подкат с навесом.

**Actua Soccer (Gremlin, 1995–98).** Голос — легендарный Барри Дэвис из BBC ([Wikipedia: Barry Davies](https://en.wikipedia.org/wiki/Barry_Davies)); сохранился даже фрагмент сценария записи ([Gremlin Archive](https://www.gremlinarchive.com/index.php/2017/04/18/actua-soccer-commentary-script/)). Банк был заметно меньше, чем у FIFA, — фразы повторялись так часто, что фанаты цитируют их наизусть 30 лет спустя ([Facebook: 90s Football Memories](https://www.facebook.com/90sFootballMemories/posts/890540083259978/)). Урок: малый банк — это не только минус, но и часть шарма ретро.

## б) Дизайн банка фраз для простейшего комментатора

Принцип из 90-х: **категория события × несколько интонационных вариантов**, анти-повтор («shuffle bag» — не играть клип, пока не прозвучали остальные в категории).

| Категория | Вариантов | Комментарий |
|---|---|---|
| Начало матча | 3 | |
| Начало 2-го тайма | 2 | |
| **Гол (обычный)** | 8–10 | Самая частая эмоция — сюда максимум |
| Гол особый (дальний / головой / пенальти) | по 2–3 | |
| Сейв вратаря | 5–6 | |
| Штанга/перекладина | 3–4 | |
| Опасный момент / удар мимо | 6–8 | Звучит чаще всего |
| Угловой | 3 | Аут можно не озвучивать вовсе |
| Офсайд / фол | 3–4 | |
| Финальный свисток (победа/ничья/разгром) | по 2–3 | |
| «Цветовые» реплики при затишье | 4–6 | Про погоду, трибуны, сезон-96 |

Итого **минимум ~60, комфортно 120–150 фраз** — этого хватает, чтобы за 10-минутный матч почти не было повторов.

**Фамилии: для v1 — не делать склейку.** Вклейка отдельных клипов-фамилий — главный источник «швов» и багов в FIFA 97. Для составов 96–99 это сотни фамилий × интонации. Вместо этого: (1) целые фразы без имён («Какой удар нападающего!»), (2) позже — по одной **целой** фразе на 2–3 звёзд каждой команды («Шевченко! Это надо было видеть!»). Если очень захочется склейку — только выкрик-фамилия сразу после гола, где шов маскирует рёв трибун.

## в) Техника

- **Формат: MP3** (или AAC/M4A). **Не OGG** — Safari на iPad не поддерживает Vorbis ([caniuse.com/ogg-vorbis](https://caniuse.com/ogg-vorbis)), а планшет — целевая платформа. Mono, 64 kbps, 22 кГц — идеально ложится в ретро-эстетику «телетрансляции».
- **Вес:** фраза ~3 с при 64 kbps ≈ 24 КБ → 150 фраз ≈ 4 МБ, 300 ≈ 8 МБ. Для браузерной игры — пустяк.
- **Web Audio API:** `fetch` → `decodeAudioData` → `AudioBuffer`, воспроизведение через `AudioBufferSourceNode`; один общий `GainNode` «канал комментатора», чтобы клипы не накладывались и приглушали музыку ([MDN: Web Audio API best practices](https://developer.mozilla.org/en-US/docs/Web/API/Web_Audio_API/Best_practices)). Важно для iPad: `AudioContext` разблокируется только жестом пользователя — `ctx.resume()` по первому тапу.
- **Нюанс памяти:** декодированный звук хранится как несжатый float32 — 300 стерео-клипов на 44 кГц съели бы ~150 МБ. Решение: mono + 22 кГц (в 4 раза меньше) и/или догрузка редких категорий по ходу матча.
- Под паттерн «CDN без сборщиков» отлично подходит [Howler.js](https://howlerjs.com/) — сам разруливает разблокировку iOS и фолбэк форматов.

## г) Тарифы ElevenLabs и стоимость банка (июль 2026)

По [elevenlabs.io/pricing](https://elevenlabs.io/pricing): **Free — $0 / 10 000 кредитов** (без коммерческой лицензии, нужна атрибуция); **Starter — $6 / 30 000**; **Creator — $22 (первый месяц $11) / 121 000**; **Pro — $99 / 600 000**. Тариф TTS: **Eleven v3 и Multilingual v2 — 1 кредит за символ**, Flash v2.5 — 0,5 ([docs](https://elevenlabs.io/docs/overview/models), [BIGVU-обзор 2026](https://bigvu.tv/blog/elevenlabs-pricing-2026-plans-credits-commercial-rights-api-costs/)).

Расчёт: 300 фраз × ~60 символов ≈ **18 000 кредитов**; с перегенерацией неудачных дублей (реально ×2–3 на v3) — **40–60 тысяч**. Вывод: **весь банк — это один месяц Creator ($11 по промо-цене) с большим запасом**, впритык хватит и Starter за $6. Для некоммерческого хобби можно растянуть на Free за 2 месяца. Подписку можно проверить прямо из Claude Code через MCP-коннектор `elevenlabs` (`check_subscription`).

## д) Русская озвучка: какая модель

- **Eleven v3** — самая выразительная модель, GA с февраля 2026, 70+ языков включая русский, лимит 5 000 символов на запрос ([elevenlabs.io/v3](https://elevenlabs.io/v3), [docs/models](https://elevenlabs.io/docs/overview/models)). Главное преимущество для комментатора — **audio-теги**: `[excited]`, `[shouts]`, `[laughs]`, `[whispers]` прямо в тексте ([блог ElevenLabs об audio tags](https://elevenlabs.io/blog/v3-audiotags)). «ГО-О-ОЛ!» с настоящим криком умеет только она.
- **Multilingual v2** — стабильнее и предсказуемее по тембру между генерациями (важно, чтобы 150 клипов звучали одним голосом), но эмоциональные пики слабее.

**Рекомендация:** v3 для эмоциональных категорий (гол, штанга, финальный свисток), при разнобое тембра в спокойных фразах — докрутить их на Multilingual v2 тем же голосом. Тексты писать по локальному скиллу `elevenlabs-v3-writer` (расстановка тегов и фонетические хаки для русского уже там). Отдельный плюс: 4 интонационных дубля одной фразы — «фишка Мотсона» из FIFA 97 — на v3 получаются бесплатно, простой перегенерацией.
