# Отчёт: бесплатные/дешёвые ассеты для браузерного «ретро-футбола» (Three.js, PS1-стиль)

Дата исследования: 17 июля 2026. Все лицензии указаны по состоянию страниц на эту дату.

## TL;DR

Полноценных «футболистов 90-х rigged + анимации» в CC0 почти нет — рабочая стратегия: **(а)** блочные CC0-персонажи Kenney/Quaternius + анимации Mixamo/Quaternius, или **(б)** игроки из примитивов кодом (проверенный паттерн в Three.js, идеально ложится в PS1-эстетику). Стадион проще собрать из itch.io-паков или сгенерировать кодом (трибуны = повторяющиеся боксы + плоские спрайты толпы). Mixamo жив и бесплатен.

## 1. Модели игроков (rigged, с анимациями)

- **Kenney — Blocky Characters v2.0** — https://kenney.nl/assets/blocky-characters — **CC0**. 20 ассетов, 18 скинов, 27 анимаций, форматы FBX/OBJ/**glTF** (glTF грузится в Three.js напрямую через GLTFLoader, без конвертации). Блочный стиль — лучший кандидат под «PS1-футболистов»: скины перекрашиваются в цвета клубов 1996–99 простой заменой текстуры-палитры.
- **Quaternius — Universal Base Characters** — https://quaternius.com/packs/universalbasecharacters.html — **CC0**, FBX/glTF/OBJ + Blend, 6 базовых humanoid-персонажей (мужские/женские, 3 типа пропорций), ~13k треугольников (для PS1-стиля можно упростить в Blender decimate-модификатором).
- **Quaternius — Universal Animation Library** — https://quaternius.com/packs/universalanimationlibrary.html — **CC0**, 120+ анимаций (бег 8 направлений, спринт, jog, падения/смерти и т.д.), совместимы с их же персонажами через retargeting. Специфических футбольных (удар, подкат) нет — их добирать в Mixamo.
- **Quaternius — Animated Human (Low Poly)** — https://opengameart.org/content/animated-human-low-poly — **CC0**, rigged, анимации idle/run/walk/jump/punch/death. Всего у Quaternius 70+ паков, 2500+ моделей, всё CC0 (https://quaternius.com/).
- **Poly Pizza** — https://poly.pizza/ — агрегатор low-poly моделей (включая архив Google Poly), только **CC0 и CC-BY**, лицензия указана у каждой модели, есть фильтр «animated» (https://poly.pizza/explore?anim=1&lic=1), форматы OBJ/FBX/glTF. Искать: soccer, football, stadium, goal.
- **OpenGameArt.org** — https://opengameart.org/ — фильтр лицензий при поиске (CC0/CC-BY). 3D-футболистов мало; полезен скорее для звуков и текстур.

## 2. Стадионы, мячи, трибуны

- **itch.io**: страница «free + low-poly + soccer» — https://itch.io/game-assets/free/tag-low-poly/tag-soccer и https://itch.io/game-assets/free/tag-football/tag-low-poly. Конкретика: **Football Stadium Pack — Lowpoly Voxel** (MrMGames, https://mrmgames.itch.io/football-stadium-pack-lowpoly-voxel), **European Stadium Pack** (torftorf1, 16 стадионов). ⚠️ На itch.io лицензия у каждого пака своя — проверять на странице (часто «free for commercial use», не всегда CC0).
- **Kenney — Sports Pack** — https://kenney.nl/assets/sports-pack — CC0, 380 ассетов, но это **2D** (top-down): пригодится для UI, иконок, мини-карты.
- Мяч и трибуны дешевле сделать кодом: мяч — IcosahedronGeometry с пиксельной текстурой, трибуны — массив BoxGeometry + плоскости с шумной «текстурой толпы» (аутентично для PS1: так делали в реальных играх 90-х).

## 3. Sketchfab в 2026 — качать ещё можно, но эпоха заканчивается

Магазин Sketchfab закрыт, платный контент переехал в Fab (Epic). **Бесплатные модели под CC-лицензиями пока остаются на Sketchfab** и доступны для скачивания; сам сайт продолжает работать как платформа бесплатного контента (https://sketchfab.com/blogs/community/sketchfab-update-what-you-need-to-know-now-that-fabs-live/). Важно: CC0/CC-BY-модели **не мигрируют в Fab** — таких лицензий там нет (https://support.fab.com/s/article/Sketchfab-to-Fab-Migration-FAQ). Вывод: нужные модели (фильтр Downloadable + License: CC0/CC-BY, тег https://sketchfab.com/tags/soccer) **скачать сейчас и хранить локально**. При CC-BY — указать автора в титрах игры.

## 4. Mixamo — жив, бесплатен, есть футбольные анимации

- Статус: Mixamo работает, бесплатен, нужен только Adobe ID, персонажи и анимации royalty-free для коммерческих проектов (официальный FAQ Adobe: https://helpx.adobe.com/creative-cloud/faq/mixamo-faq.html, обзор 2026: https://toolradar.com/tools/mixamo).
- В библиотеке есть именно футбольные клипы: Soccer Idle, Soccer Pass, Soccer Penalty Kick, Soccer Header, Soccer Tackle (подтверждение набора: https://superhivemarket.com/products/realmotion-pro-blender-addon-for-animation-, туториалы: https://www.youtube.com/watch?v=8duql12oG9I).
- **Пайплайн Mixamo → Blender → glTF** (канонический гайд Don McCurdy: https://www.donmccurdy.com/2017/11/06/creating-animated-gltf-characters-with-mixamo-and-blender/): персонаж FBX в T-pose → каждая анимация FBX «without skin» → импорт в Blender, анимации в NLA-стек → экспорт одним .glb → Three.js AnimationMixer. Практический гайд под Three.js: https://sbcode.net/threejs/gltf-animations/ и https://github.com/funwithtriangles/blender-to-threejs-export-guide.
- **Без Blender вообще**: https://mixamo2gltf.com/ — онлайн-склейка Mixamo FBX в один GLB (стоит проверить качество результата; для новичка-оркестратора это短ейший путь).
- Ограничение Mixamo: свои модели он риггает только гуманоидов, а Mixamo-риг к Kenney-блочным персонажам придётся ретаргетить (в Blender — аддоны retargeting), либо загрузить блочную модель прямо в авто-риггер Mixamo.

## 5. Игроки из примитивов кодом — рекомендуемый вариант «А»

Прецеденты подтверждают уместность:
- **Squad Goals** — Three.js футбол, персонажи собраны из BoxGeometry с vertex colors, стиль Crossy Road (https://www.seeles.ai/games/sports/squad-goals-3d-soccer-game).
- **Pixel Ball** — браузерный 4v4 футбол на Three.js (https://discourse.threejs.org/t/pixel-ball-a-three-js-football-game/92658).
- Гайд «PS1 style graphics in Three.js» — Roman Liutikov (https://romanliutikov.com/blog/ps1-style-graphics-in-threejs): низкополигональность — база стиля + vertex snapping и affine-текстуры.

Оценка: для PS1/CRT-эстетики примитивы **уместнее** «гладких» готовых моделей — реальные футболисты ISS/FIFA 96–98 и были ~200–500 полигонов. Анимации при этом кодом (процедурный бег через качание конечностей-боксов) — просто, и полностью в стиле. Это ещё и снимает вопрос лицензий и веса файлов для планшетов.

## 6. Ретро-шрифты для табло

- **Press Start 2P** — https://fonts.google.com/specimen/Press+Start+2P — OFL, битмап 8-бит, использовать кратными 8px размерами.
- **Jersey 10** (и семейство Jersey 15/20/25) — https://fonts.google.com/specimen/Jersey+10 — OFL, Sarah Cadigan-Fried, стилизация под LED-табло 90-х — идеально для счёта/таймера.
- **DSEG** — https://github.com/keshikan/DSEG — OFL 1.1, честный 7-/14-сегментный шрифт «как на стадионном табло», 50+ вариантов.
- Дополнительно на Google Fonts (все OFL): VT323 (терминал), Silkscreen (пиксель).

## 7. Звуки: толпа, свисток, стадион

- **Freesound.org** — https://freesound.org/browse/tags/stadium/ — в поиске включить фильтр лицензии **CC0** (иначе попадаются CC-BY и CC-BY-NC; NC для игры с монетизацией нельзя). Искать: stadium ambience, crowd cheer, referee whistle, crowd chant.
- **Mixkit** — https://mixkit.co/free-sound-effects/crowd/ — бесплатно, собственная лицензия Mixkit (коммерческое использование разрешено).
- **ZapSplat** — пак «Football Stadium Crowd» https://www.zapsplat.com/sound-effect-packs/football-stadium-crowd/ — бесплатно с указанием авторства (или Gold-подписка без указания).
- Совет по стилю: пропустить звуки через lowpass-фильтр/bitcrusher (Web Audio API) — «телевизионный» звук 90-х дёшево усиливает CRT-эстетику.
